## 安装使用

- 获取项目代码

```bash
git clone https://gitee.com/TaiAi/electron-vue3-quiet.git
```

- 安装依赖

```bash
cd electron-vue3-quiet

npm i -g @antfu/ni

ni
```

- 运行(开发)

```bash
nr dev
```

- 打包(生产)

```bash
nr build
```
